Overview

EA Forex Automation is a MetaTrader 5 Expert Advisor designed to automate trading during high-volatility news events. It uses pending buy/sell stop orders with adjustable lot sizes and trailing stop mechanisms to manage trades efficiently. The system includes license key verification to restrict unauthorized access and an Auto Trade mode with dynamic lot calculation based on account balance.

Main Functions

  • License verification system to block unauthorized users
  • Auto placing 5 levels of BuyStop and SellStop orders
  • Customizable lot sizes for each level
  • Dynamic lot calculation in Auto Trade mode
  • Built-in Take Profit (TP) and Trailing Stop system
  • Automatic order cancellation when price retreats
  • Balance-based risk control (optional)
  • Trading logs for review and analysis

Tech Stack

  • Platform: MetaTrader 5
  • Language: MQL5
  • Data Source: Economic Calendar / Broker Server
  • Environment: Desktop (Windows-based)
